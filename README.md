# Running Instructions: Angry Birds 1 vs 1 Two Player Game

This project is a competitive multiplayer game built with the **Pygame CE** framework[cite: 1, 10]. Follow the steps below to set up your environment and launch the game.

---

## 1. Prerequisites
Before running the project, ensure your system meets the following requirements:
* **Python**: You must have Python installed on your machine.
* **Pip**: Ensure the Python package installer (pip) is available to install dependencies.

---

## 2. Installation Steps

### Step 1: Install Python 
If you do not have Python installed, download it from the official website:
> [python.org/downloads](https://www.python.org/downloads/) 

### Step 2: Install Dependencies
The game relies on the `pygame-ce` (Community Edition) library for graphics, events, and audio. Open your terminal or command prompt and run:
```bash
pip install pygame
```

---
## 3. How to Launch the Game

To start the **Angry Birds 1 vs 1** application, follow these steps: 

1.  **Open your terminal** and navigate to the project's root directory. 
2.  **Execute the following command** to start the application: 

```bash
python3 main.py
```
